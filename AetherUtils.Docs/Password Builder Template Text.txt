Password Builder Template Text

Commands

1. [p]='<prefix text>' -> denotes the prefix text for the password.
	Example
	[p]='Pre'  -> denotes to add the text 'Pre' to the beginning of the password.

2. [s] -> denotes a special character.
	Example:
	[s]='!@#$' -> denotes a special character selected only from the supplied list.
	[s] 	   -> denotes a special character selected from the default special list.

3. [fn]='<firstName' -> denotes the first name of the user. If no option specified, defaults to lower case.
	Options:
	1. [uU]pper -> denotes to use upper case.
	2. [lL]ower -> denotes to use lower case.
	
	Example:
	[fn:upper]='John' -> denotes to use the first name 'John' with an upper case letter.

4. [ln]='lastname' -> denotes the last name of the user. If no option specified, defaults to lower case.
	Options:
	1. [uU]pper -> denotes to use upper case.
	2. [lL]ower -> denotes to use lower case.

	Example:
	[ln:lower]='Smith' -> denotes to use the last name 'Smith' with a lower case letter.

5. [d]='<date>' -> denotes a date.

	Example:
	[d] 	   	 -> adds the current year.
	[d]='2021' 	 -> adds the year 2021.
	[d]='10-15-2021' -> adds the date 10-15-2021 to the password.

6. [i] -> denotes a digit character.
	Example:
	[i]='15769' -> adds a digit character selected only from the supplied list.
	[i]	    -> adds a digit character selected from the default digit list (1234567890).


7. [suff]='<suffix text>' -> denotes the suffix text for the password.
	Example:
	[suff]='Suf' -> denotes to add the text 'Suf' to the end of the password.

Full Examples (The following are just examples of what might be generated based on the template. Every generated password will be random based on supplied template parameters.)

1. [p]='Test'[s][i][fn:lower]='John'[s][ln:upper]='Smith'[d] 	=> Test$5j#S2024
2. [s][fn:upper]='Doug'[ln:lower]='Smurf'[s][i][d][suff]='end'	=> *Ds%82024end
3. [p]='TEM'[s][fn:lower]='Bob'[ln:upper]='Williams'[s][d]	=> TEM^bW(2024
4. [fn]='William'[s]='@*-'[ln]='Johnson'[s]'%&!'		=> w-j!
